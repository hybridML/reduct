Reasoning Capabilities

∀

Universal Rules

ALL(X, Y) + IS(a, X) → IS(a, Y)

"All physicians have clinical access" + "Dr. Evans is physician" → Dr. Evans has clinical access

→

Transitivity

ALL(X, Y) + ALL(Y, Z) → ALL(X, Z)

"Physicians → clinical access → PHI access → audit required" — full chain derived

⇒

Conditionals

IF(X, Y) + IS(a, X) → IS(a, Y)

"If on-call then emergency_access" + "Nurse Kelly is on-call" → emergency access

¬

Negation-as-Failure

Cannot prove IS(a, X) → NOT_IS(a, X)

"No record that Bob is clinical staff" → Bob does not have clinical access (default-deny)

⏱

Temporal

AFTER(N, X) + IS(a, X) → HOLDS(a, expires_after_N)

"After 90 days, terminated" + "Alice is terminated" → access expires

#

Cardinality

AT_MOST(N, role), EXACTLY(N, role)

"At most 2 approvers" + 3 people have approver → violation flagged

⊞

Mutual Exclusivity

MUTEX(A, B) — no entity holds both

"Clinical and billing access are mutually exclusive" + Alice has both → violation

⊕

Contradiction Detection

IS(a, X) + NOT_IS(a, X) → CONTRADICTION

"Alice has access" + "Alice does not have access" → flagged immediately

Get Started

Install & Run
git clone https://github.com/hybridML/reduct.git
cd reduct
pip install -r requirements.txt

# Start the API server
uvicorn api.server:app --reload --port 8000

# Or run the CLI demo
python pipeline.py

Configure Your Domain

Edit config/domain.yaml with your policies, entities, and constraints. Or use the built-in HIPAA config.

Query

POST to /reason with a natural language query. Get back derived conclusions, provenance chains, and constraint violations.

Audit

Every derivation includes its full provenance chain. Trace any conclusion back to source facts and the inference rule used.
